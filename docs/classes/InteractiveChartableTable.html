    The physical table to be proxied

  • options Array

    An array of options objects from which to generate the collection of datatables. See jQuery.autoChart module for documentation of the options structure and default values.

  • onCharted Function

    A callback to be invoked when a charting button is clicked. The function should take an array of selectedfield indices as an argument.

  • onSeriesSelected Function

    A callback to be invoked when a series is selected for charting in multi-chartable tables. The function should take an array of selected field indices as an argument. Returning false from the callback disables all unchecked checkboxes. Returning true form the callback enables all disabled checkboxes, and returning an array of indices disables all checkboxes at the associated field indices.
