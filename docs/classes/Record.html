Record Class

Extends DataCollection
Defined in: js\jquery.autochart.js:951
Module: jQuery.autoChart

A collection of cells that represent a record. Records can be one of two types:

  1. Header Records: Describe the data contained in the record. These headers can contain categories, series descriptions, unit descriptions, and table descriptions.
  2. Data Records: Contain category values that correlate to the data contained in value fields.

Records contain two sub collections (arrays):

  1. Category Cells: These cells contain the category for the record (year, region, ect.).
  2. Value Cells: These cells contain values for the record.
