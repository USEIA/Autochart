DynamicClass Class

Defined in: js\jquery.autochart.js:236
Module: jQuery.autoChart

Base class for most classes in the module. It allows for classes that take an object as a constructor argument, and apply a set of the properties from the object to the class instance. If the properties are not found on the object, then the instance will pass hasOwnProperty for the missing properties, with undefined values.
